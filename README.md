## Go WhatsApp Rest

### Run App
```sh
    git clone git@github.com:geekbim/Go-WhatsApp-Rest.git
    cd Go-WhatsApp-Rest
    cp .env.example .env
    go mod tidy
    sh run-service.sh
```

### Run Test
```sh
    sh run-test.sh
```
